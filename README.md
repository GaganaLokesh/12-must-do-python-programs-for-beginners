# 🐍 Python Beginner Programs (1–12)

This repository contains 12 beginner-friendly Python programs designed to build a strong foundation in programming concepts.

12 essential Python programs for beginners covering basics, loops, data structures, OOP and file handling.

---

## 📚 Concepts Covered

* Variables & Input/Output
* Conditional Statements
* Loops
* Functions
* Lists & Dictionaries
* File Handling
* Exception Handling
* Object-Oriented Programming (OOP)
* Inheritance

---

## 📂 Programs List

### 1. Simple Calculator

* Performs basic operations (+, -, *, /)
* Concepts: Input/Output, Operators, if-else

---

### 2. Number Guessing Game

* User guesses a random number
* Concepts: Loops, Conditions, `random` module

---

### 3. Prime Number Checker

* Checks whether a number is prime
* Concepts: Functions, Loops

---

### 4. List Manager

* Add, delete, and display items
* Concepts: Lists, CRUD operations

---

### 5. Word Frequency Counter

* Counts frequency of words in a sentence
* Concepts: Strings, Dictionary

---

### 6. Student Grade System

* Assigns grades based on marks
* Concepts: Lists, Tuples, Conditions

---

### 7. Contact Book

* Add, search, and delete contacts
* Concepts: Dictionary, CRUD operations

---

### 8. Notes App 📂

* Add, view, delete, and search notes
* Concepts:

  * File handling (`with open`)
  * File modes (`r`, `a`, `w`)
  * Loops and string handling
  * f-strings and `strip()`

---

### 9. Safe Calculator ⚠️

* Performs operations with error handling
* Concepts:

  * `try-except`
  * Handling `ZeroDivisionError`
  * Handling `ValueError`

---

### 10. Bank Account System 🏦

* Deposit, withdraw, and check balance
* Concepts:

  * Classes & Objects
  * Constructor (`__init__`)
  * Methods and `self`

---

### 11. Inheritance Example 🚗

* Parent and child class implementation
* Concepts:

  * Inheritance
  * Method overriding
  * `super()` and constructor chaining

---

### 12. To-Do List ⭐

* Task manager with file storage
* Concepts:

  * Functions
  * File handling
  * Menu-driven program

---

## 🎯 Key Learnings

* Strong understanding of Python fundamentals
* Hands-on practice with real-world problems
* Improved problem-solving skills
* Introduction to OOP concepts used in real applications

2. Navigate to the project folder

